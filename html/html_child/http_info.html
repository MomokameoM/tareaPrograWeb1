HTTP

HTTP, acrónimo en inglés de HyperText Transfer Protocol («protocolo de transferencia de hipertexto»), es un protocolo de comunicación utilizado para la transferencia de datos en la World Wide Web. Este protocolo permite la comunicación entre los navegadores web y los servidores web, facilitando la solicitud y entrega de recursos como páginas web, imágenes, archivos, entre otros. HTTP es uno de los protocolos fundamentales que hacen posible el funcionamiento de la web moderna.
